<img width="397" height="882" alt="image" src="https://github.com/user-attachments/assets/6108d947-d287-47b9-8995-d31fd2021ce0" /># 🤖 AskAi Application

A modern Android chat application built with **Jetpack Compose**, **MVVM architecture**, and **Firebase**.  
The app supports chat sessions, history management, authentication, and profile customization.

---

## 🚀 Features

- 🔐 Firebase Authentication (Login / Register)
- 💬 AI Chat interaction
- 📚 Chat session history
- 👤 User profile with editable username(not image profile edit due to firebase storage is not free)
